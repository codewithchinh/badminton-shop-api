# Requirements

## MVP Scope

The first version will focus on:

- Customers can browse products.
- Customers can filter products by brand and category.
- Customers can view product details and variants.
- Customers can add products to cart.
- Customers can checkout with COD payment.
- Customers can view order history.
- Staff can confirm and process orders.
- Operator can update shipping status.
- Admin can manage products, brands, categories, and variants.

## Later Features

- Return requests
- Warranty requests for rackets and shoes
- Promotions
- Revenue dashboard
- Smart racket and shoe recommender
