# Badminton Shop API
Backend API for an online badminton equipment store.

## Tech Stack
- ASP.NET Core Web API
- PostgreSQL
- Entity Framework Core
- JWT Authentication
- Swagger / OpenAPI

## Main Features
- Product catalog
- Product variants for rackets, shoes, and accessories
- Shopping cart
- Checkout and order management
- Return and warranty requests
- Role-based management for customer, staff, operator, director, and admin
